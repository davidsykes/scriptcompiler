ScriptCode
    A pointer to a unit of compiled script
    ScriptCode* script_code = script_code_create(scriptData);

ScriptInstance
    A runnable instance of a Script function
        It contains
            A pointer to the unit of script
            A current position within the script
            A variable stack for processing
            Local variables
    ScriptInstance* inst = script_instance_create(script_code);

VariableCollection
    A collection of script variables
    VariableCollection* globals = variable_collection_create();

FnRoutine
    A pointer to the function to handle fn routines.
    typedef void (*FN_ROUTINE)(const char* name, VariableValue* parameters, VariableValue* fn_return_value);
        name                fn routine name
        parameters          The parameters for the routine
        parameter_count     The number of parameters supplied
        fn_return_value     The variable value to use for the return value



    It can be repeatedly run, returning 0 for pauses mid script, and 1 when the script completes.
