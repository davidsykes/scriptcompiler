

ScriptCode is a pointer to a unit of compiled script

ScriptCodeNavigator contains a ScriptCode pointer, and allows navigation of the script

ScriptInstance is a runnable instance of a Script function
    It contains
        A pointer to the unit of script
        A current position within the script
        A variable stack for processing
     local variables, and maintains it's position.
It can be repeatedly run, returning 0 for pauses mid script, and 1 when the script completes.
